# Quizzler

A true/false trivia quiz app built with Python and Tkinter.

## Features
- Fetches random questions from the [Open Trivia Database](https://opentdb.com/)
- Visual feedback with green/red background on correct/incorrect answers
- Tracks and displays your score
- Disables buttons when the quiz is finished

## Requirements
- Python 3
- `requests` library

## How to run
1. Clone the repository
2. Install dependencies: `pip install requests`
3. Run `main.py`
